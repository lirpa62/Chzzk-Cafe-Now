# 치즈 카페 나우 - Chzzk Cafe Now

<div align="center">

**네이버 카페에서 치지직 클립을 바로 보는 가장 가벼운 방법**

</div>

네이버 카페 게시글에 첨부된 치지직 클립 링크를 감지해 OG 링크 썸네일 대신 게시글 안에서 바로 재생할 수 있는 치지직 플레이어를 표시하는 Chrome 확장 프로그램입니다.

치지직 클립 링크를 누르기 위해 새 탭으로 이동하지 않아도 됩니다.  
카페 게시글 안에서 플레이어를 바로 열고 원본 클립 링크도 함께 확인할 수 있습니다.

---

## 주요 기능

- **치지직 클립 OG 카드 변환**: 네이버 카페의 치지직 클립 OG 카드 썸네일을 iframe 플레이어로 자동 교체합니다.
- **단독 클립 링크 변환**: OG 카드 없이 클립 링크만 있는 경우에도 OG 카드형 플레이어로 자동 교체합니다.
- **중복 링크 정리**: OG 카드와 같은 클립을 가리키는 단독 링크 컴포넌트는 중복 표시되지 않도록 제거합니다.
- **제목 정리**: 치지직 API에서 스트리머명과 클립 제목을 조회해 `스트리머명 - 제목 (클립 링크)` 형식으로 표시합니다.
- **카드 폭 확장**: 네이버 카페 기본 OG 카드보다 넓은 최대 `800px` 너비로 표시합니다.
- **동적 페이지 대응**: 네이버 카페의 동적 로딩과 내부 프레임 환경에서도 동작합니다.
- **업데이트 안내 배너**: 확장 프로그램 업데이트 후 열린 네이버 카페 탭에 새로고침 안내 배너를 표시합니다.
- **원본 링크 유지**: 카드 제목 영역을 클릭하면 원본 치지직 클립으로 이동할 수 있습니다.

## 지원 링크

현재 공식 임베드 플레이어로 안정적으로 재생 가능한 치지직 **클립** 링크만 지원합니다.

```text
https://chzzk.naver.com/clips/{클립ID}
https://chzzk.naver.com/embed/clip/{클립ID}
```

다음 주소는 다시보기/페이지 전체 주소라서 iframe에 영상만 표시되지 않으므로 지원하지 않습니다.

```text
https://chzzk.naver.com/video/{비디오ID}
https://naver.me/{단축ID}
```

## 로컬 설치

1. Chrome에서 `chrome://extensions/`를 엽니다.
2. 오른쪽 위의 **개발자 모드**를 켭니다.
3. **압축해제된 확장 프로그램을 로드합니다**를 누릅니다.
4. 이 프로젝트 폴더를 선택합니다.
5. 치지직 클립 링크가 포함된 네이버 카페 게시글을 새로고침합니다.

테스트용 클립 URL:

```text
https://chzzk.naver.com/clips/06B56r5cXb
```

## 배포 ZIP 생성

Chrome 확장 프로그램 배포용 ZIP은 다음 명령으로 생성합니다.

```bash
npm run package
```

생성된 파일은 `dist/chzzk-cafe-now-{version}.zip`에 저장됩니다. 배포 ZIP에는 `manifest.json`, `README.md`, `src/`, manifest에서 참조하는 아이콘 파일만 포함되며, `tests/`, `scripts/`, `package.json`, 원본 `assets/icon.png`는 제외됩니다.

Firefox 애드온 배포용 ZIP은 다음 명령으로 생성합니다.

```bash
npm run package:firefox
```

생성된 파일은 `dist/chzzk-cafe-now-firefox-{version}.zip`에 저장됩니다. Firefox용 ZIP은 같은 소스 파일을 사용하지만, manifest의 background 선언을 Firefox가 지원하는 `background.scripts` 형식으로 변환하고 `browser_specific_settings.gecko` 값을 추가합니다.

## 동작 방식

확장 프로그램은 네이버 카페 문서와 내부 프레임에서 `div.se-component.se-oglink`에 포함된 치지직 클립 링크를 찾습니다. 링크가 발견되면 OG 링크 카드의 썸네일 영역만 공식 치지직 임베드 주소를 사용하는 iframe으로 교체합니다.

```text
https://chzzk.naver.com/embed/clip/{클립ID}
```

OG 카드가 없고 링크만 단독으로 존재하는 경우에는 네이버 카페 OG 카드와 비슷한 구조를 생성한 뒤 그 안에 iframe 플레이어와 원본 링크 정보를 함께 표시합니다.

## 참고 사항

- 치지직 클립이 삭제되었거나 비공개 상태라면 플레이어가 정상 재생되지 않을 수 있습니다.
- 네이버 카페나 치지직의 DOM 구조가 변경되면 일부 변환 로직이 동작하지 않을 수 있습니다.
- 다시보기/동영상 페이지는 공식 클립 임베드와 다르게 페이지 전체가 iframe에 표시되므로 지원 대상에서 제외했습니다.
- 확장 프로그램을 수정한 뒤에는 `chrome://extensions/`에서 확장 프로그램을 새로고침해야 변경 내용이 반영됩니다.
